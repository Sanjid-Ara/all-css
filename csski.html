CSS এর পূর্ণরূপ হলো Cascading Style Sheets.


সি এস এস এর মাধ্যমে এইচটি এম এল ফাইলের বিভিন্ন উপাদান গুলোকে সুন্দর ভাবে সাজাতে সক্ষম হই। কেননা এইচটি এম এল এর কিছু ( Attributes ) দিয়ে সামান্য কিছু স্টাইল করতে পারি। কিন্তু সি এস এস দিয়ে নিজেদের তৈরিকৃত ওয়েব সাইটটি বিভিন্নি ধরনের ফন্ট, কালার, মার্জিন, প্যাডিং, ব্যাকগ্রাউন কালার সহ সুন্দর ভাবে নির্ধারণ করতে পারি ।


সিএসএস এর কাজ হলো ওয়েব পেজটিকে আকর্ষিত করা বা ডিজাইন বা Layout তৈরী করা।HTML এবং CSS এই দুটি দিয়েই আপনি পুরো ওয়েবসাইট এর ডিসাইন করতে পারবেন। এক কথায় একটি সুন্দর ওয়েবসাইট ডিসাইন করতে হলে আপনাকে এই CSS Language খুব ভালো করে শিখতে হবে।
